Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passa o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Parágrafo
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Quebra de linha
  • Listas númeradas
    • Use type para mudar o parâmetro
      • 1, a, A, i, I
  • Listas com demarcadores
    • Use type para mudar o parâmetro
      • disc, square, circle
  • Listas de definição
    • Use com a lista de difinição com as tag <dt> e <dd>
  • Links
    • Parametros
      • <target>
        • _blank
        • _self
        • _parent
      • <rel>
        • external
        • nofollow
  • Citação simples
  • Citação completas
  • Texto deletado
  • Texto inserido
  • Marcação
  • Destaque
  • Ênfase
  • Texto grande
  • Texto pequeno
  • Sobrecrito
  • Subcrito
